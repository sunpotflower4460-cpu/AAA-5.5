# Phase 5 Arrange Report

## 目的

MVPのUI/UXを、残心らしい静けさ・余白・現代和の方向へ磨き込み,体験も独自に最大化する。

## 改善した内容

- 和紙のような柔らかい背景、淡い紙目、控えめな円相の透かしを追加した。
- ヒーロー、カード、空状態の角丸・罫線・影を調整し、現代和の静かな第一印象に寄せた。
- 検索バー、フィルター、FAB、カードのhover / active / focus表現を、押しやすさを保ちながら控えめに磨いた。
- メモカードに細い刃のような縦線、金の小さな印、静かな「ひらく」導線を追加した。
- エディタの保存ステータスに微細な余韻表現を加え、保存中だけ静かに呼吸する表示へ調整した。
- iPhone幅で上部余白、カード高さ、保存ステータス、操作ボタンが詰まりにくいよう微調整した。
- 動きを苦手とする環境へ配慮し、`prefers-reduced-motion` でアニメーションを抑える指定を追加した。

## 変更した主なファイル

- `src/index.css`
- `src/components/NotesList.tsx`
- `src/components/NoteCard.tsx`
- `src/components/NoteEditor.tsx`

## デザイン判断

装飾を増やすのではなく、背景の紙目、薄い円相、細い線、金の印だけで「残心」らしさを出した。余白と本文の読みやすさを優先し、カードやボタンの影は控えめにして、iPhoneで開いた瞬間に静かで美しい印象になるよう調整した。検索や新規作成は見つけやすくしつつ、書く体験の主役であるタイトルと本文入力を邪魔しないようにした。

## 守ったこと

- 複雑にしすぎない
- 余白を守る
- 侘び寂びを意識する
- iPhone-first
